Задача

Имеется файл EmpImp.RData. (Примечание: файлы с расширением .RData используются для сохранения в них workspace. Workspace --- это совокупность имеющихся в данный момент переменных и их текущих значений.) В сохранённом в этот файл workspace имеется единственная переменная --- ExpImp, в ней хранится датафрейм, содержащий для каждого из субъектов РФ (и не только субъектов) данные по экспорту, импорту различных видов грузов.
Необходимо:
  • Создать датафрейм, в котром будет информация лишь по субъектам РФ.
  • В этот датафрейм добавить две колонки: одна --- с суммарным экспортом по субъекту, другая --- с суммарным импортом.
  • Создать функцию, которая возвращает субъекты, в которых суммарный экспорт превосходит суммарный импорт.

Реализация

Функция get_subjects_df() решает первую задачу --- создаёт датафрейм, где будет информация лишь по субъектам РФ.
Функции get_export_sums_col(df_subjects) и get_import_sums_col(df_subjects) принимают датафрейм с информацией по субъектам РФ и создают столбцы соответственно суммарного экспорта и импорта, которые с помощью функции cbind можно присоединить к датафрейму субъектов.
Функция get_where_export_sum_greater(df_subjects, export_sum_col_ind, import_sum_col_ind) принимает
  • датафрейм с информацией по субъектам РФ, в который добавлены столбцы суммарного экспорта, импорта,
  • индексы столбцов суммарного экспорта, импорта
и возвращает вектор имён субъектов, в которых суммарный экспорт превосходит суммарный импорт.
