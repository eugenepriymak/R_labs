Задача

Проводится медицинское исследование вакцины: после вакцинации каждый день фиксируется температура пациента, и так на протяжении нескольких дней. Файл data.RData содержит единственную переменную --- data. Она представляет собой list из data.frame-ов, содержащих столбцы id (идентификаторы пациентов) и term (температуры пациентов). (Примечание: файлы с расширением .RData используются для сохранения в них workspace. Workspace --- это совокупность имеющихся в данный момент переменных и их текущих значений.)
Необходимо написать функцию get_id, которая создаёт data.frame, содержащий id пациентов и их среднюю за все дни температуру. При этом нужно учесть, что некоторые пациенты в какие-то дни могут пропустить проверку температуры, поэтому для чистоты исследования таких пациентов не надо включать в этот data.frame.

Реализация

Назовём интересующий нас data.frame со средними температурами df_mean_temp. В функции get_id он инициализируется значениями из data.frame-а первого дня, далее мы итерируемся по остальным дням, ищем пациента из df_mean_temp в data.frame этого дня, и если не находим, то удаляем строку из df_mean_temp с помощью функции delete_row_from_df, а если находим, то прибавляем его температуру за этот день к тому, что хранится в df_mean_temp (когда мы обрабатываем data.frame последнего дня, мы после этого прибавления сразу делим результат на количество дней).
В результате df_mean_temp содержит средние температуры, притом только для пациентов, фиксировавших её без пропусков.
